# FaceMorph

基于opencv+Dlib的面部合成（Face Morph）
这里的面部合成指的的是把一张脸逐渐的变化成另外一张脸。
具体见[我的博客](http://blog.csdn.net/wangxing233/article/details/51549880 "悬停显示")
